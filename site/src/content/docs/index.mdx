## What this is

A curated set of **principles, patterns, and philosophical alignment** for how cognitive work composes into durable artifacts. The claims are substrate-agnostic: they're justified by how attention, memory, and authority *work*, not by which editor or AI tool you use.

This is the **trunk** of a larger system. Two downstream repositories specialize these foundations into opinionated implementations — see [related repos](#related-repos) below.

## Three classification axes

Every principle and pattern on this site lives at the intersection of three orthogonal axes. One-sentence crib:

> **Tier asks *who*. Stratum asks *how portable*. Tier-of-abstraction asks *what's assumed*.**

| Axis | Answers | Where |
|---|---|---|
| **Tier** (1 / 2 / 3) | WHO is this for? | Organizational sections in a repo (kernel / stack / work). This trunk repo publishes Tier 1 content only. |
| **Stratum** (1–5) | HOW MUCH of this transfers if copied? | [Five strata](/knowledge-work-foundations/principles/07-five-strata/) — colored badge under each page title |
| **Tier of abstraction** (0 / 1 / 2) | WHAT substrate does this claim assume? | [Tiers of abstraction](/knowledge-work-foundations/principles/00-tiers-of-abstraction/) — zero-tier = paper+pencil, tier-1 = digital, tier-2 = LLM-agentic |

The axes are **lenses, not taxonomies** — real artifacts slide between strata and tiers. The labels help you weight *how stable is this advice* without pretending certainty.
